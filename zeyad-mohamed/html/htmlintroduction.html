Welcome to your first step in HTML.

HTML is an abbreviation for HyperText Markup Language.
It is the first and main language in the front-end development track, which is responsible for the main structure of any website.
It is not considered as a programming language.

The commands in this language are in tags < > containing elements. Some elements contain attributes.
The difference between those definitions is that whatever is in the tags are elements but everything but the main word or command is an attribute. For example:

< img src= "hello.png" >

The tag is: < img src= "hello.png" >

The element is: img src= "hello.png"

The attribute is: src= "hello.png"

Some tags have an opening and closing, while others do not. This is according to whether the tag contains content or is content. In the <img> tag, it is content, while in the <html> tag it contains content.
