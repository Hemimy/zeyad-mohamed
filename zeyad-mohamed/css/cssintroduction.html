Welcome to your first step in CSS.

CSS is an abbreviation for Cascading Style Sheets. It is the 2nd language in the front-end development track, which is responsible for the decoration and style of websites.

The commands in this language are in this form:

.divclass{
property: value;
}

, this form:

element{
property: value;
}

or this form:

.divclass element{
property: value;
}
